# Student Attendance System (SAS)

A Python-based face recognition system for tracking student attendance.

Download the latest `.exe` (1.2) from [Releases](https://github.com/Kxress/Student-Attendance-System/releases).

### Keyboard shortcuts:

Alt + P – Choose folder to save attendance list

Alt + C – Enter the class name

Alt + T – Set the maximum minutes a person is considered present
